# GD UUID

GD UUID is a Godot library that generates UUIDs.
Currently, it supports UUID v7 only.

[Godot Asset Library](https://godotengine.org/asset-library/asset/)

## Demo

You can try it [here](https://hotaritobu.github.io/gd-uuid/).
